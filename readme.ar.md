# MarkdownTranslator
[[English]](readme.md)

<div dir=rtl>

مكتبة للغة الأسس للتحويل من صيغة ماركداون إلى صيغة HTML.

## الاستخدام

* أضف المكتبة للمشروع:

```
اشمل "مـحا"؛
مـحا.اشمل_ملف("Alusus/MarkdownTranslator"، "مـترجم_ماركداون.أسس")؛
```

<div dir=ltr>

```
import "Apm";
Apm.importFile("Alusus/MarkdownTranslator");
```

</div>

* أنشئ متغيرا من صنف المترجم:


```
عرف مترجم: مـترجم_ماركداون؛
```

<div dir=ltr>

```
def translator: MarkdownTranslator;
```

</div>

* حول من صيغة ماركداون إلى HTML:


```
نص_إتش_ت_م_ل = مترجم.ترجم(نص_ماركداون)؛
```

<div dir=ltr>

```
htmlString = translator.translate(markdownString);
```

</div>

دالة `ترجم` تستلم `نـص` وترجع `نـص`.

### دالة عند_الترميز (onHtmlTag)

يمكن التحكم بعملية إنشاء عناصر الترميز عبر تحديد الدالة المغلفة `عند_الترميز` (`onHtmlTag`) والتي تستلم ثلاث معطيات:

* نوع_عنصر_الترميز: مـؤشر_محارف.
  سلسلة محارف لنوع عنصر ترميز HTML. على سبيل المثال: h1.

* النص_الأصلي1: نـص.
  النص الأصلي المستخرج من ملف ماركداون. لا يشمل هذا النص محارف ترميز ماركداون. مثلا، نص العنوان لن يحتوي على
  الرمز #.

* النص_الأصلي2: نـص.
  النص الأصلي الثاني في حال كان العنصر يحتاج لنصين كما هو الحال مع الروابط، حيث تحتاج الروابط إلى النص المقروء
  بالإضافة إلى نص العنوان المستهدف.

ترجع هذه الدالة النص المرمز. إن أرجعت نصًا فارغًا فسيُنشئ مـترجم_ماركداون النص المرمز بنفسها.
